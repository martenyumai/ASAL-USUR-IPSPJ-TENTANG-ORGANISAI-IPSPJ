# ASAL-USUR-IPSPJ-TENTANG-ORGANISAI-IPSPJ
Asal Usul IPSPJ:Menggali Lebih Dalam Akar Organisasi Pelajar Papua di Jawa Timur IPSPJ, atau Ikatan Pelajar dan Siswa Papua Jawa Timur, merupakan sebuah organisasi yang memiliki peran penting bagi pelajar dan siswa asal Papua yang menempuh pendidikan di Jawa Timur.
